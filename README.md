#  **Проект модуля. Нейросеть для предсказания калорийности блюд**  


## Описание задачи
Проект направлен на создание и обучение нейросети, которая предсказывает калорийность блюд по трём типам данных:  

1. **Ингредиенты** – в виде списка, закодированного с помощью эмбеддингов и self-attention.  
2. **Изображение блюда** – извлечение признаков с помощью ResNet50.  
3. **Числовые параметры** – масса блюда и количество ингредиентов.  

Модель может быть интегрирована в фитнес- и health-приложения для автоматической оценки калорийности блюд.

## Архитектура модели

Модель использует мульти-модальный подход:  

1. **Ингредиенты**  
   - `nn.Embedding` для кодирования ингредиентов  
   - `nn.MultiheadAttention` для выявления важности каждого ингредиента  
   - `nn.Linear` для проекции в скрытое пространство (`HIDDEN_DIM`)  

2. **Изображение**  
   - Предобученный `ResNet50` без классификационного слоя (`num_classes=0`)  
   - Линейная проекция признаков в скрытое пространство  

3. **Числовые признаки**  
   - Масса блюда и количество ингредиентов проходят через MLP (`num_proj`)  
   - Масштабирование признаков для балансировки с другими эмбеддингами  

4. **Фьюзинг**  
   - Поэлементное умножение различных признаков  
   - Конкатенация всех исходных и смешанных признаков  
   - Итоговая регрессия через MLP (`regressor`) для предсказания калорий
